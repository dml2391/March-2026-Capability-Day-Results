# March-2026-Capability-Day-Results
Some sample calculations carried out during the ARCHER2 Capability Day sessions of March 3rd/4th 2026.
The two smaller calculations, for t=6.96 times 10^32 and t=9.17 times 10^32, were run during the pre-Capability Day of March 3rd using 512 nodes
(maximum job time 1hr). Both these calculations required longer than 1hr and so were restarted (see comments at the end of 
the respective output files). The three calculations for t>10^33 were carried out on the Capability Day of March 4th and used 1024 nodes
(maximum job time 3hrs). All three of these runs completed in under the 3hrs maximum. The precision setting was set to the standard value eps=0.005 
for each calculation of Z(t).
