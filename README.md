# March-2026-Capability-Day-Results
Some sample calculations carried out during the ARCHER 2 Capability Day sessions of March 3rd/4th 2026
The two smaller calculations for t=6.96*10^32 and t=9.17*10^32 were run on the pre-Capability Day of March 3rd using 512 nodes
(maximum job time 1hr). Both these calculations required longer than 1hr and so were restarted (see comments at the end of 
each output file). The three calculations for t<10^33 were carried out on the Capability Day of March 4th and used 1024 nodes
(maximum job time 3hrs). All three calculations completed in under 3hrs.
